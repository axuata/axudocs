# `LengthUnit`
`LengthUnit` is a TypeScript type that defines various units of length for conversion.

## 🔧 Supported Units

### 📏 Metric Units
| Unit |     Description     |
|:----:|:-------------------:|
| `nm` | Nanometer (100⁻⁹ m) |
| `μm` | Micrometer (10⁻⁶ m) |
| `mm` | Millimeter (10⁻³ m) |
| `cm` | Centimeter (10⁻² m) |
| `m`  |  Meter (Base Unit)  |
| `km` |  Kilometer (10³ m)  |

### 📐 Imperial Units
| Unit |  Description   |
|:----:|:--------------:|
| `in` | Inch (2.54 cm) |
| `ft` |  Foot (12 in)  |
| `yd` |  Yard (3 ft)   |
| `ch` | Chain (66 ft)  |
| `mi` | Mile (5280 ft) |

### ✨ Special Units
| Unit  |         Description         |
|:-----:|:---------------------------:|
| `ly`  | Light-year (\~9.46×10¹² km) |
| `nmi` |   Nautical Mile (1852 m)    |
